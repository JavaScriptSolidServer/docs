# Installation

## Prerequisites

- Node.js 18+

## Install from npm

```bash
npm install -g javascript-solid-server
```

## Install from source

```bash
git clone https://github.com/JavaScriptSolidServer/JavaScriptSolidServer
cd JavaScriptSolidServer
npm install
npm link
```

## Verify installation

```bash
jss --help
```
